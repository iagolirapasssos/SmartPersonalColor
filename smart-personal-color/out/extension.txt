## <kbd>Events:</kbd>
**SmartPersonalColor** has total 2 events.

### 1. AnalysisResult
Fires when analysis is complete. Returns average face RGB, undertone, 12-season result, 15 named color palettes (YailList of YailLists: index 0 = name, index 1-5 = hex colors), and a transparent PNG path of the face with background removed (no external API).

| Parameter | Type
| - | - |
| r | number
| g | number
| b | number
| undertone | text
| season | text
| palettes | list
| faceImagePath | text

### 2. Error
Fires when analysis fails. Provides a descriptive error message.

| Parameter | Type
| - | - |
| message | text
